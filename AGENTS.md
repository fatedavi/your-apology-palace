# AGENTS.md — Your Apology Palace

## Stack

React 19, Vite 7, TypeScript 5.8, TanStack Router + React Query, TailwindCSS v4, shadcn/ui (New York), Framer Motion, ESLint, Prettier, Bun (local), Vercel deploy.

**Client-only SPA** — no SSR, no backend, no persistence.

## Commands

| Command | Action |
|---|---|
| `npm run dev` | Start Vite dev server |
| `npm run build` | Production build → `dist/` |
| `npm run preview` | Preview production build |
| `npm run lint` | ESLint all files |
| `npm run format` | Prettier write all files |

No test or typecheck scripts exist.

## Must-know constraints

- **`routeTree.gen.ts` is auto-generated by `@tanstack/router-plugin`** — never edit manually. `.prettierignore` excludes it. Deleted file triggers regeneration on next build/dev.
- **`server-only` imports are banned.** Use `*.server.ts` naming convention (enforced by ESLint).
- **No backend/persistence** — all state lives in React component state only.
- **`@` alias** maps to `./src/*` via `vite-tsconfig-paths` + `tsconfig.json`.

## Architecture

```
src/
  main.tsx               — App entry: renders <App /> from router.tsx
  router.tsx             — TanStack Router + QueryClient + App component
  routeTree.gen.ts       — auto-generated (do not edit)
  routes/
    __root.tsx           — root layout, head/meta, error/404 components
    index.tsx            — single route "/" → renders ForgiveMeApp
  components/
    forgive/             — app scenes (loading → name → question → final)
    ui/                  — shadcn/ui primitives
  lib/
    utils.ts             — `cn()` helper (clsx + tailwind-merge)
  styles.css             — Tailwind v4 source + OKLCH palette + custom utilities
public/img/              — static images (capoo gifs/jpg)
```

The app is a scene-based state machine: `loading → name → question → final` (see `ForgiveMeApp.tsx`). Single route `/`.

## Deploy (Vercel)

`vercel.json` is pre-configured:
```
outputDirectory: "dist"
rewrites: all routes → /index.html
```
Build runs `npm run build` → Vite outputs to `dist/` → Vercel serves static SPA.

## Dev workflow quirks

- **Bun is the local package manager** (bun.lock, bunfig.toml), but deploy uses npm (vercel.json build command).
- **`bunfig.toml` enforces a 24h supply-chain guard** (`minimumReleaseAge`). `@lovable.dev/vite-tanstack-config` was exempted but is no longer used.
- **TypeScript is lenient:** `noUnusedLocals`, `noUnusedParameters` off; `verbatimModuleSyntax` off; `@typescript-eslint/no-unused-vars` off.
- **shadcn/ui** style: `new-york`, `@/components/ui/` alias, `@/lib/utils` for `cn()`, `@/hooks` for hooks.
